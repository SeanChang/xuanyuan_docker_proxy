## 配置说明

爱快路由内置 Docker 功能，可在路由器上直接运行容器应用。由于爱快自带的默认 Docker 镜像源稳定性较差，甚至会出现故障，强烈建议配置轩辕镜像专属域名作为加速源。

| 爱快版本 | 界面特征 | 配置入口 |
| --- | --- | --- |
| **4.0**（推荐） | 全新 UI，左侧「高级服务 → Docker」，顶部有「镜像仓库 / 本地镜像」等标签；**仅 64 位系统支持 Docker** | 镜像仓库 → 镜像源 |
| **3.x**（旧版） | 经典界面，左侧「高级应用 → 插件管理」；**32 位系统仍支持 Docker** | 插件管理 → 添加 → 镜像库下载 |

- **爱快 4.0：** 高级服务 → Docker → 镜像仓库 → 镜像源
- **爱快 3.x：** 高级应用 → 插件管理 → 添加 → 镜像库下载

> 请根据您的爱快版本选择下方对应的配置教程。如果您使用的是 4.0 版本，请直接查看「爱快 4.0」章节，按截图逐步操作即可。

> **爱快 4.0 升级注意事项：** 根据 [爱快官方公告](https://www.ikuai8.com/support/cjwt/2021-10-15-08-37-02/4-0.html)，**爱快 4.0 的 32 位系统不支持 Docker 和应用市场**。若您当前为 32 位系统且需要使用 Docker，**请勿升级到 4.0**，可继续留在 3.x 并按「爱快 3.x」章节配置镜像源；若计划升级到 4.0，请先将系统切换为 **64 位**（软路由用户建议使用 64 位系统进行刷机）。

## 爱快 4.0

> **系统要求：** 本教程适用于爱快 4.0 **64 位**系统。32 位系统不支持 Docker 和应用市场——若您为 32 位且需使用 Docker，请勿升级到 4.0；如需升级，请切换至 64 位系统。详见 [爱快 4.0 升级注意事项](https://www.ikuai8.com/support/cjwt/2021-10-15-08-37-02/4-0.html)。

> **为什么要配置？** 爱快 4.0 自带的默认 Docker 镜像源稳定性较差，部分节点甚至会出现故障，导致镜像搜索或下载失败。我们已实测确认：在爱快 4.0 中配置轩辕镜像专属域名后，可以正常搜索、下载 Docker 镜像。

> 开始之前，请先 [登录](https://xuanyuan.cloud/dashboard?next=%2Fusage%2Fikuai) 轩辕镜像控制台，在左侧「专属域名」菜单中复制您的**专属域名**（格式如 `***.xuanyuan.run`）。爱快 4.0 推荐使用专属域名，**无需填写用户名和密码**。

### 一、进入镜像源设置

1. 登录爱快路由管理后台，在左侧菜单栏找到 **「高级服务」**，点击展开后选择 **「Docker」**。
2. 在 Docker 管理页面顶部的标签栏中，点击 **「镜像仓库」**。
3. 点击页面右上角的蓝色按钮 **「镜像源」**，打开镜像源设置窗口。

![爱快 4.0 进入 Docker 镜像源设置：高级服务 → Docker → 镜像仓库 → 镜像源](https://img.xuanyuan.dev/docker/ikuai/ikuai-1.png)

图 1：依次点击「Docker」→「镜像仓库」→「镜像源」（对应步骤 ① ② ③）

### 二、添加轩辕镜像源

1. 在弹出的 **「镜像源设置」** 窗口中，点击右上角的 **「添加」** 按钮。
2. 在 **「添加镜像源」** 弹窗中，左侧协议下拉框保持默认的 **「https://」** 即可，无需修改。
3. 在右侧输入框中填写您的专属域名，**只填域名本身，不要带 https:// 前缀**，例如：

   ```
   ***.xuanyuan.run
   ```

4. 在 **「备注」** 输入框中填写 **「轩辕镜像」**。
5. 点击 **「安装」** 按钮，完成本条镜像源的添加。

![爱快 4.0 添加轩辕镜像源：选择 https 协议并填写专属域名](https://img.xuanyuan.dev/docker/ikuai/ikuai-8.png)

图 2：协议保持 https://，输入框只填专属域名（不带 https://），备注填「轩辕镜像」，点击「安装」

> **填写注意：** 协议下拉框已默认选中 `https://`，右侧输入框**只需填写域名**（如 `***.xuanyuan.run`），**不要在输入框中重复填写 https://**，也不要在末尾添加多余斜杠。

### 三、调整优先级并保存

爱快自带的默认镜像源稳定性较差，甚至会出现故障。添加「轩辕镜像」后，**必须将其拖动到列表第一位**，确保优先使用轩辕镜像加速拉取。

1. 添加成功后，在 **「镜像源设置」** 列表中可以看到「轩辕镜像」条目。爱快会**按从上到下的顺序依次尝试**各个镜像源，请按住左侧的六点拖拽图标，将「轩辕镜像」**拖动到列表最顶部**。
2. 确认「轩辕镜像」位于第一位后，点击窗口左下角的 **「保存」** 按钮。

![爱快 4.0 将轩辕镜像拖动到列表第一位并保存](https://img.xuanyuan.dev/docker/ikuai/ikuai-2.png)

图 3：由于默认镜像源不稳定，需将「轩辕镜像」拖到列表第一位（步骤 ④），然后点击「保存」（步骤 ⑤）

### 四、确认重启 Docker

1. 点击「保存」后，系统会弹出提示：「修改镜像源配置需要重启 docker，服务重启过程中容器与插件应用暂不可用，是否重启？」
2. 点击 **「确定」**，等待 Docker 服务重启完成，新配置即可生效。

![爱快 4.0 保存镜像源后确认重启 Docker](https://img.xuanyuan.dev/docker/ikuai/ikuai-3.png)

图 4：保存后弹出重启确认框，点击「确定」重启 Docker（步骤 ⑥）

> **重启提示：** Docker 重启期间，正在运行的容器和插件应用会短暂中断。建议在业务低峰期操作，或提前停止重要容器。

### 五、验证配置是否成功

配置完成并重启 Docker 后，建议立即下载一个常用镜像来验证加速是否生效。以下以 nginx 为例：

1. 回到 **「镜像仓库」** 页面，在左上角搜索框中输入 **「nginx」** 并搜索。
2. 在搜索结果中找到官方 nginx 镜像，点击卡片上的 **「下载」** 按钮。

![爱快 4.0 搜索 nginx 镜像并点击下载](https://img.xuanyuan.dev/docker/ikuai/ikuai-4.png)

图 5：搜索 nginx（步骤 ⑦），点击「下载」按钮（步骤 ⑧）

3. 在弹出的 **「下载镜像」** 窗口中，从 **「镜像标签」** 下拉框选择需要的版本（如 `1.30` 或 `latest`）。
4. 点击 **「确定」** 开始下载。

![爱快 4.0 选择镜像标签并确认下载](https://img.xuanyuan.dev/docker/ikuai/ikuai-5.png)

图 6：选择镜像标签（步骤 ⑨），点击「确定」开始下载（步骤 ⑩）

5. 切换到顶部 **「本地镜像」** 标签，再点击 **「下载中」** 子标签，即可看到镜像正在下载（状态显示「下载中……」）。

![爱快 4.0 本地镜像下载中状态](https://img.xuanyuan.dev/docker/ikuai/ikuai-6.png)

图 7：在「本地镜像 → 下载中」查看下载进度（步骤 ⑪）

6. 等待下载完成后，切换回 **「本地镜像」** 标签，即可在列表中看到已下载的镜像（如 nginx 1.30），表示配置成功。

![爱快 4.0 镜像下载成功](https://img.xuanyuan.dev/docker/ikuai/ikuai-7.png)

图 8：镜像下载成功，配置验证通过

> **配置成功！** 看到镜像出现在「本地镜像」列表中，说明轩辕镜像源已正常工作。之后您在「镜像仓库」中搜索并下载的任何镜像，都会优先通过轩辕镜像加速拉取。

## 爱快 3.x

> **版本说明：** 以下步骤适用于爱快 **3.x** 经典界面（含 32 位系统）。爱快 3.x 的 32 位系统仍可正常使用 Docker 插件；若您已升级至 4.0，请切换到「爱快 4.0」章节（4.0 的 32 位系统不支持 Docker）。

爱快 3.x 通过 Docker 插件的「镜像库下载」功能拉取镜像。请先参考 [爱快 Docker 官方文档](https://www.ikuai8.com/support/ymgn/lyym/2019-12-13-06-35-05/2021-03-08-07-27-41.html) 完成 Docker 插件的安装。

> 配置前请先 [登录](https://xuanyuan.cloud/dashboard?next=%2Fusage%2Fikuai) 轩辕镜像控制台，在左侧「专属域名」菜单中复制您的**专属域名**（如 `***.xuanyuan.run` 或 `***.xuanyuan.dev`，以控制台显示为准）。

### 一、进入插件管理并添加容器

1. 登录爱快路由管理后台，在左侧菜单栏找到 **「高级应用」**，点击展开后选择 **「插件管理」**。
2. 点击页面上方的 **「添加」** 按钮，进入新建插件页面。
3. 在 **「上传方式」** 中选择 **「镜像库下载」**。

![爱快 3.x 插件管理镜像库下载：在镜像搜索框输入专属域名/镜像名](https://img.xuanyuan.dev/docker/ikuai/ikuai.jpg)

### 二、通过专属域名搜索并下载镜像

1. 在 **「镜像搜索」** 输入框中，按格式填写：

   ```
   专属域名/镜像名
   ```

   例如：

   ```
   ***.xuanyuan.run/nginx
   ```

   或

   ```
   ***.xuanyuan.dev/qbittorrent
   ```

2. 点击 **「搜索」** 按钮，等待搜索结果列表加载。
3. 在搜索结果中找到目标镜像，点击对应行的 **「进入下载页」**。
4. 在下载页选择镜像标签并完成下载安装。

> **搜索格式说明：** 爱快 3.x 不支持像 4.0 那样单独配置镜像源列表，需在搜索框中直接带上专属域名前缀。格式为 `专属域名/镜像名`，中间用斜杠分隔，不要加 `https://` 前缀。

### 三、验证配置是否成功

若搜索能返回结果且镜像下载安装成功，说明专属域名配置正确。之后每次拉取新镜像，均在「镜像搜索」中使用相同的 `专属域名/镜像名` 格式即可。

> **配置成功！** 镜像安装完成后可在插件管理列表中看到对应容器。如遇 3.7.22 版本镜像源异常，请参阅下方「常见问题」章节。

## 常见问题

### Q：爱快 4.0 找不到 Docker 入口？

爱快 4.0 的 **32 位系统不支持 Docker 和应用市场**。若您为 32 位系统，请勿升级到 4.0，可留在 3.x 使用 Docker；如需使用 4.0 的 Docker 功能，请切换至 **64 位** 系统。详见 [爱快官方 4.0 升级注意事项](https://www.ikuai8.com/support/cjwt/2021-10-15-08-37-02/4-0.html)。

### Q：爱快 4.0 配置后仍然拉取失败？

1. 确认协议下拉框为 `https://`，输入框只填域名（**不要**在输入框中带 https://）
2. 确认「轩辕镜像」已拖动到列表**最顶部**（优先级最高）
3. 确认保存后已点击「确定」重启 Docker 服务
4. 确认专属域名复制正确，且账户仍有可用流量

### Q：爱快 3.x 已配置镜像仓库，但拉取镜像失败或镜像源无法生效？

> 爱快路由系统 **3.7.22** 版本的 Docker 插件在加载自定义镜像源时存在已知缺陷，可能导致镜像仓库地址保存后仍无法正常使用。该问题与轩辕镜像服务本身无关，属于爱快固件侧的行为。

1. **降级系统版本（推荐）：** 将固件降级至 **3.7.19**，该版本下 Docker 镜像源配置可正常生效。
2. **使用离线上传方式：** 在其他设备上拉取镜像后导出为 tar 包，上传至爱快路由导入。
3. **升级到 4.0：** 如条件允许，升级至爱快 4.0 后按新版教程配置，体验更稳定。
