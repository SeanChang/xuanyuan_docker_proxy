# it-tools

面向开发者和IT从业者的实用工具集。[查看详情](https://it-tools.tech)。

项目仓库：https://github.com/CorentinTh/it-tools

## Docker部署

使用以下命令快速部署it-tools容器：

```bash
docker run -d -p 8080:80 --name it-tools -it corentinth/it-tools
```

参数说明：
- `-d`：后台运行容器
- `-p 8080:80`：将容器内的80端口映射到主机的8080端口
- `--name it-tools`：指定容器名称为it-tools
- `-it`：提供交互式终端支持
