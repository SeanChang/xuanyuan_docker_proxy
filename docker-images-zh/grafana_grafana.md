# Grafana Docker 镜像


## 运行 Grafana Docker 容器

启动 Docker 容器时，需将 Grafana 服务绑定到外部端口 3000。执行以下命令：

```bash
docker run -d --name=grafana -p 3000:3000 grafana/grafana
```

服务启动后，默认管理员账号密码为 `admin/admin`，可直接登录使用。
