# Dify.AI 前端 Docker 镜像

Dify 是一款 LLM 应用开发平台，目前已有超 10 万款应用基于 Dify.AI 构建。它整合了后端即服务（BaaS）与 LLMOps 理念，覆盖构建生成式 AI 原生应用所需的核心技术栈，包括内置 RAG 引擎。借助 Dify，你可基于任意 LLM 自建类 Assistants API 和 GPTs 的能力。
