# Bitnami Secure Image for kserve-localmodel-controller

## 镜像概述和主要用途

Bitnami Secure Image for kserve-localmodel-controller 是 Bitnami 提供的针对 kserve-localmodel-controller 的安全加固镜像。该镜像目前已不再通过 Docker Hub 免费提供，而是作为商业订阅的 OCI 制品，提供 Debian 和 Photon 两种基础操作系统格式。其主要用途是为 kserve-localmodel-controller 的部署提供安全增强的运行环境，满足企业级安全需求。


## 核心功能和特性

升级至 Photon 基础操作系统的 Bitnami Secure Images 可带来以下核心优势：

- **硬加固安全镜像**：基于流行开源软件构建，实现接近零漏洞（Near-Zero Vulnerabilities）的安全基线  
- **漏洞管理能力**：支持漏洞分类与优先级排序，提供 VEX 声明、KEV（已知被利用漏洞）和 EPSS（漏洞利用预测评分系统）评分  
- **合规性支持**：聚焦合规需求，提供 FIPS、STIG 认证及离线部署选项，包含安全物料清单（SBOM）  
- **供应链安全溯源**：通过 in-toto 实现软件供应链来源证明，确保制品完整性  
- **Helm Charts 兼容性**：对主流 Helm Charts 提供一流支持，与原 Debian 基础镜像的 Helm Charts 完全兼容  


## 使用场景和适用范围

该镜像适用于对安全性、合规性和供应链可追溯性有严格要求的企业级环境，特别是：  
- 需要部署 kserve-localmodel-controller 并满足行业安全标准（如 FIPS、STIG）的场景  
- 注重开源软件漏洞管理和风险控制的组织  
- 要求完整软件供应链溯源能力的关键业务系统  
- 依赖 Helm Charts 进行标准化部署的 Kubernetes 环境
