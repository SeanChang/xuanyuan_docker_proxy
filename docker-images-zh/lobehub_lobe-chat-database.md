# Lobe Chat Database Edition


## 概述  
Lobe Chat 是一款开源的聊天机器人框架，主打可扩展性（支持函数调用）与高性能，支持用户一键免费部署个人专属的 /LLM 网页应用。


## 核心特点  
- **开源透明**：代码完全开放，支持自由查看、修改及二次开发，无使用限制。  
- **灵活扩展**：内置函数调用（Function Calling）能力，可按需扩展功能，适配多样化场景。  
- **高效运行**：框架经过性能优化，确保聊天交互流畅，响应速度快。  
- **简易部署**：简化部署流程，无需复杂配置，快速搭建个人 LLM 网页应用。
