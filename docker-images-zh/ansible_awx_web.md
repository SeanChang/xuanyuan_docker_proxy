# Ansible AWX（旧版本）

## 镜像概述

本镜像为Ansible AWX的旧版本镜像，目前已不再维护。AWX是Ansible自动化平台的Web界面，提供任务调度、配置管理、监控等功能。请注意，若需安装或升级至12.0.0及以上版本，应使用官方维护的`ansible/awx`镜像（https://hub.docker.com/r/ansible/awx）。

## 注意事项

- **不再维护**：本镜像针对旧版AWX，官方已停止更新和支持，可能存在安全漏洞或功能缺陷。
- **版本限制**：仅适用于12.0.0以下的AWX版本，不支持12.0.0及更高版本的部署需求。

## 适用范围

仅推荐在需要维持现有旧版AWX环境（12.0.0以下版本）的场景中使用，不建议用于新部署。
