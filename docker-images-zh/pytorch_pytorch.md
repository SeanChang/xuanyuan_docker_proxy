### PyTorch 简介  

PyTorch 是一款以 Python 为优先的深度学习框架。  

它在 Python 环境中提供张量（Tensors）和动态神经网络支持，并具备强大的 GPU 加速能力。  

框架支持动态计算图，常用于计算机视觉、自然语言处理等场景的模型训练与推理。通过 Docker 容器可统一 CUDA 与依赖环境，便于在开发、测试与生产集群中复现运行环境。
