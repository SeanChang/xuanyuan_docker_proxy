# Apache Pulsar 简介


## 什么是 Apache Pulsar？  
[Apache Pulsar]([]) 是一款高可扩展、低延迟的消息平台，可运行在通用硬件上。它支持基于主题的简单发布-订阅语义，提供消息至少一次投递保证，支持订阅者自动游标管理，并具备跨数据中心复制能力。
