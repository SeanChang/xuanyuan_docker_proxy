## 关于 Portainer  
Portainer 是一款轻量级管理界面工具，可帮助你轻松管理 Docker 主机或 Swarm 集群。  

其设计初衷是部署和使用都尽可能简单——仅需单个容器即可运行在任意 Docker 引擎上（支持 Linux 和 Windows 版 Docker）。通过 Portainer，你可以管理 Docker 栈、容器、镜像、卷、网络等资源，同时兼容独立 Docker 引擎和 Docker Swarm 集群。
