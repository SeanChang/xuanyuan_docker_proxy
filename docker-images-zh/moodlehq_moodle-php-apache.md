## й…ҚзҪ®

### Apacheй…ҚзҪ®

жӯӨй•ңеғҸдҪҝз”ЁApache HTTPDжңҚеҠЎеҷЁжҸҗдҫӣжүҖжңүеҶ…е®№пјҢйңҖиҰҒжңҖе°‘зҡ„жүӢеҠЁй…ҚзҪ®гҖӮ

еҸҜд»ҘдҪҝз”Ё`APACHE_DOCUMENT_ROOT`зҺҜеўғеҸҳйҮҸй…ҚзҪ®Apacheзҡ„`DocumentRoot`жҢҮд»ӨпјҢдҫӢеҰӮпјҡ

```bash
docker run \
    --name web0 \
    -p 8080:80 \
    -v $PWD/moodle:/srv/moodle
    -e APACHE_DOCUMENT_ROOT=/srv/moodle \
    moodle-php-apache:latest
```

жіЁж„ҸпјҡжҢҮе®ҡ`DocumentRoot`е°ҶиҰҶзӣ–й»ҳи®Өж №зӣ®еҪ•пјҢе№¶дјҡйҳ»жӯўй•ңеғҸиҮӘеҠЁй…ҚзҪ®д»»дҪ•Moodleзү№е®ҡй…ҚзҪ®гҖӮ

### PHPй…ҚзҪ®

дҪңдёәе®Ңж•ҙPHPй…ҚзҪ®ж–Үд»¶зҡ„иҪ»йҮҸзә§жӣҝд»Јж–№жЎҲпјҢжӮЁеҸҜд»ҘеңЁеҗҜеҠЁе®№еҷЁж—¶жҢҮе®ҡдёҖз»„еёҰеүҚзјҖзҡ„зҺҜеўғеҸҳйҮҸпјҢиҝҷдәӣеҸҳйҮҸе°ҶиҪ¬жҚўдёәiniж јејҸй…ҚзҪ®гҖӮ

д»»дҪ•еҗҚз§°д»Ҙ`PHP_INI-`дёәеүҚзјҖзҡ„зҺҜеўғеҸҳйҮҸе°Ҷиў«з§»йҷӨеүҚзјҖпјҢе№¶еңЁдё»е‘Ҫд»ӨеҗҜеҠЁеүҚж·»еҠ еҲ°ж–°зҡ„iniж–Үд»¶дёӯгҖӮ

```bash
docker run \
    --name web0 \
    -p 8080:80 \
    -v $PWD/moodle:/var/www/html
    -e PHP_INI-upload_max_filesize=200M \
    -e PHP_INI-post_max_size=210M \
    moodle-php-apache:latest
```

## еҲқе§ӢеҢ–и„ҡжң¬

жӯӨй•ңеғҸж”ҜжҢҒдҪҝз”Ё`docker-entrypoint.d`зӣ®еҪ•зҡ„иҮӘе®ҡд№үеҲқе§ӢеҢ–и„ҡжң¬гҖӮиҝҷдәӣи„ҡжң¬еҸҜд»ҘжҳҜд»ҘдёӢж јејҸпјҡ

* йқһеҸҜжү§иЎҢзҡ„`.sh`и„ҡжң¬пјҢе°Ҷиў«_жәҗд»Јз ҒеҢ–_е№¶ж”№еҸҳеҪ“еүҚдёҠдёӢж–Ү
* еҸҜжү§иЎҢзҡ„`.sh`и„ҡжң¬пјҢе°ҶеңЁеҪ“еүҚдёҠдёӢж–Үдёӯ_жү§иЎҢ_
* `.ini`ж–Үд»¶пјҢе°Ҷиў«еӨҚеҲ¶еҲ°PHPй…ҚзҪ®зӣ®еҪ•пјҲ`/usr/local/etc/php/conf.d`пјү

ж ҮеҮҶеҢ…еҗ«д»ҘдёӢи„ҡжң¬пјҡ

* `10-wwwroot.sh` - йқһеҸҜжү§иЎҢи„ҡжң¬пјҢз”ЁдәҺеңЁжңӘжҸҗдҫӣ`APACHE_DOCUMENT_ROOT`ж—¶зҢңжөӢиҜҘеҖј

иҝҷдәӣи„ҡжң¬ж— жі•еҲ йҷӨпјҢдҪҶеҸҜд»ҘйҖҡиҝҮеңЁжӮЁиҮӘе·ұзҡ„`docker-entrypoint.d`дҪҚзҪ®еҲӣе»әе…·жңүеҢ№й…Қж–Үд»¶еҗҚзҡ„ж–Үд»¶жқҘзҰҒз”Ёе®ғд»¬гҖӮ

иҝҳеҸҜд»ҘжҸҗдҫӣе…¶д»–и„ҡжң¬пјҢдҫӢеҰӮпјҢиҰҒй…ҚзҪ®PHPд»Ҙж”ҜжҢҒжӣҙй«ҳзҡ„`upload_max_filesize`йҖүйЎ№пјҢжӮЁеҸҜд»Ҙе°Ҷд»ҘдёӢеҶ…е®№ж·»еҠ еҲ°`config/10-uploads.ini`ж–Үд»¶дёӯпјҡ

```
; жҢҮе®ҡдёҠдј зҡ„жңҖеӨ§ж–Үд»¶еӨ§е°Ҹдёә200M
upload_max_filesize = 200M
post_max_size = 210M
```

еҗҜеҠЁе®№еҷЁж—¶пјҢеҸҜд»Ҙдј е…Ҙй…ҚзҪ®зӣ®еҪ•пјҡ

```
docker run \
    --name web0 \
    -p 8080:80 \
    -v $PWD/moodle:/var/www/html
    -v $PWD/config:/docker-entrypoint.d \
    moodle-php-apache:latest
```

иҝҷдәӣеҲқе§ӢеҢ–ж–Үд»¶е°ҶжҢүз…§еҪ“еүҚеҢәеҹҹи®ҫзҪ®е®ҡд№үзҡ„жҺ’еәҸеҗҚз§°йЎәеәҸжү§иЎҢпјҢй»ҳи®Өдёәen_US.utf8гҖӮ

#### pcov

еҗҜеҠЁе®№еҷЁж—¶жҢҮе®ҡд»ҘдёӢзҺҜеўғеҸҳйҮҸеҸҜеҗҜз”Ёе®ғпјҡ

```bash
PHP_INI-pcov.enabled=1
```

## еҸӮи§Ғ

жӯӨе®№еҷЁжҳҜMoodleејҖеҸ‘е®№еҷЁйӣҶзҡ„дёҖйғЁеҲҶпјҢеҸҰиҜ·еҸӮи§Ғпјҡ

* [moodle-docker](https://github.com/moodlehq/moodle-docker) - еҹәдәҺdocker-composerзҡ„е·Ҙе…·йӣҶпјҢеҸҜйӣ¶й…ҚзҪ®иҝҗиЎҢMoodleејҖеҸ‘зҺҜеўғ
* [moodle-db-mssql](https://github.com/moodlehq/moodle-db-mssql) - дёәMoodleејҖеҸ‘й…ҚзҪ®зҡ„LinuxзүҲMicrosoft SQL Server
* [moodle-db-oracle](https://github.com/moodlehq/moodle-db-oracle) - дёәMoodleејҖеҸ‘й…ҚзҪ®зҡ„Oracle XE
