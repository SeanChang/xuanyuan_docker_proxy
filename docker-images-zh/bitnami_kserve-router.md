# 核心功能与特性

我们建议考虑从早期 Debian 镜像升级到基于 Photon 的 Bitnami 安全镜像。它们与原始 Debian 镜像使用的热门 Helm 图表完全兼容，升级后用户将获得以下优势：

- 流行开源软件的硬化安全镜像，具有接近零漏洞特性
- 漏洞分类与优先级划分，包含 VEX 声明、KEV 和 EPSS 评分
- 聚焦合规性，支持 FIPS、STIG 和离线选项，包括安全物料清单（SBOM）
- 通过 in-toto 提供软件供应链来源证明
- 对互联网热门 Helm 图表的一流支持
