# 精简化Ubuntu Python镜像  

本文介绍由Canonical提供的基于Ubuntu的Python Docker镜像。该镜像会接收安全更新，并滚动升级至更新的Python或Ubuntu版本。**本仓库可免费使用，且不受用户速率限制。**


## 关于Python  

Python是一种高级通用编程语言，其设计理念强调代码可读性，采用显著的缩进格式。Python是动态类型语言，支持垃圾回收，兼容结构化、面向对象、函数式等多种编程范式。更多信息可访问[python.org]([])。  

本镜像是精简化的Ubuntu基础镜像，仅包含Python运行时及其标准库，可直接用于生产环境，也可作为基础镜像添加应用所需的额外依赖。

### 通道标签说明  
通道标签按稳定性从高到低排序：`stable`（稳定）、`candidate`（候选）、`beta`（测试）、`edge`（边缘）。更具风险的通道默认可用（例如，若列出`beta`，则`edge`也可使用；若列出`stable`，则四个通道均可用）。镜像会按`edge→beta→candidate→stable`的顺序发布，确保稳定性逐步提升。

## 使用方法  

### 本地启动镜像  
```sh
docker run -d --name python-container -e TZ=UTC ubuntu/python:3.13-25.04_stable
```


### 测试与调试  
#### 查看容器日志  
```sh
docker logs -f python-container
```

#### 进入交互式Python解释器  
```bash
docker exec -it python-container python3
```
