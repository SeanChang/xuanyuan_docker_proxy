# Dify.AI 后端 Docker 镜像  

Dify 是一款 LLM 应用开发平台，目前已有超过 10 万款应用基于 Dify.AI 构建。它整合了后端即服务（BaaS）与 LLMOps 的核心理念，覆盖构建生成式 AI 原生应用所需的全技术栈，包括内置的 RAG 引擎。借助 Dify，你可以基于任意大语言模型（LLM）自建类似 Assistants API 和 GPTs 的能力。
