### 标签更新机制
这些标签会在新的次要版本或补丁版本发布时自动更新。相关更新由 `src/docker_python_nodejs` 目录下的 Python 脚本处理，该脚本通过 GitHub Actions 每日运行两次。


### 支持平台
镜像标签支持 linux/amd64 和 linux/arm64 平台，alpine 变体仅支持 linux/amd64（详见 [issue #70] ）。

## 常用操作
### 基础命令
```bash
# 从 Docker Hub 拉取镜像
docker pull docker.xuanyuan.run/nikolaik/python-nodejs:latest

# 从 GitHub 构建镜像
docker build -t nikolaik/python-nodejs github.com/nikolaik/docker-python-nodejs

# 运行镜像
docker run -it docker.xuanyuan.run/nikolaik/python-nodejs bash
```


### 作为基础镜像使用
```Dockerfile
FROM docker.xuanyuan.run/nikolaik/python-nodejs:latest

USER pn  # 所有镜像默认包含用户 pn（uid 1000，gid 1000）
WORKDIR /home/pn/app
```
