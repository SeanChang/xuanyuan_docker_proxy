# Blinko - 开源自托管灵感笔记工具

## 关于 Blinko  
Blinko 是一款开源自托管工具，专为需要快速捕捉、整理瞬间灵感的人设计。它能让你在想法闪现时即刻记录，不让任何创意火花溜走——无论是随手记、待办项还是临时灵感，都能轻松收纳。


## 核心功能  

### AI 增强笔记检索  
借助 Blinko 的 RAG（检索增强生成）技术，你能用自然语言直接搜索笔记。比如输入“上周记的产品方案”，系统会快速定位相关内容，找笔记不再翻来翻去。  

### 数据自主掌控  
隐私至关重要。所有笔记和数据都安全存储在你的自托管环境中，没有第三方介入——你完全掌控自己的信息，不用担心数据泄露。  

### 高效速记体验  
想法来了？即刻记录。笔记以纯文本格式存储，打开就能写；支持完整 Markdown 语法，排版、分享都顺手，不用纠结复杂操作。  

### 轻量架构，强劲性能  
基于 Next.js 构建，Blinko 兼顾“轻量”与“强性能”：界面简洁不臃肿，运行速度快，日常使用流畅无卡顿。  

### 开放协作，共同成长  
作为开源项目，Blinko 的代码在 GitHub 完全透明。你可以提交代码、提改进建议，或参与功能讨论——社区的每一份力量，都在让它变得更好。
