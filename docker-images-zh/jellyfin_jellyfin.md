# Jellyfin  


## 免费软件媒体系统  


Jellyfin 是一套免费软件媒体系统，让你能完全掌控媒体文件的管理与流媒体播放。它定位为 Emby、Plex 等专有媒体系统的替代方案，支持通过专用服务器向各类终端设备（手机、电脑、电视等）的客户端应用推送媒体内容。  


### 核心特点  
- **无附加限制**：不绑定付费授权，没有隐藏功能或商业套路，所有功能完全开放。  
- **跨平台支持**：基于 .NET Core 框架开发（从 Emby 3.5.2 版本衍生并移植而来），可在 Windows、Linux、macOS 等主流系统运行。  
- **社区驱动**：由一群开发者协作维护，目标是打造更贴合用户需求的媒体系统，欢迎所有感兴趣的人参与共建。
