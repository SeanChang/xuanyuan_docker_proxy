# Chainguard MinIO 镜像文档

## 镜像概述和主要用途
Chainguard MinIO镜像是低至零CVE（常见漏洞和暴露）的容器镜像，旨在帮助用户构建、交付和运行安全软件。该镜像提供MinIO对象存储服务的安全部署选项，适用于对安全性要求较高的环境。

## 核心功能和特性
- 低至零CVE漏洞：通过严格的供应链安全措施，最大限度减少安全风险
- 多版本支持：提供包括FIPS合规版本在内的多种变体，满足不同安全标准需求
- 完整安全元数据：包含软件物料清单（SBOMs）、安全公告及漏洞信息，提升透明度与可审计性

## 使用场景和适用范围
适用于需要安全可靠对象存储服务的场景，包括企业级数据存储、云原生应用后端、数据备份与恢复等。特别适合对安全性、合规性有严格要求的组织和项目。

## 下载和使用方法

### 下载镜像
该镜像可从Docker Hub和Chainguard Registry获取：

#### 从Docker Hub拉取：
```bash
docker pull chainguard/minio:latest
```

#### 从Chainguard Registry拉取：
```bash
docker pull cgr.dev/chainguard/minio:latest
```

### 注意事项
- 若遇到速率限制问题，建议使用Chainguard Registry拉取镜像
- 有关额外版本（含FIPS支持）、SBOMs、安全公告和漏洞信息，可访问[Chainguard镜像目录的minio页面](https://images.chainguard.dev/directory/image/minio)
