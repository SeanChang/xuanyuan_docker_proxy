# Bitnami Secure Image for suitecrm

当前，Bitnami Secure Image for suitecrm 已不再通过 Docker Hub 免费提供。该镜像作为构建好的 OCI 制品，支持 Debian 和 Photon 两种基础操作系统格式，需通过 Bitnami Secure Images 商业订阅获取。不过，Docker Hub 上仍提供其他 Bitnami Secure Images 的开发者版本，完整列表可通过访问 [公共应用目录]  并筛选“Availability Type: Trial”获取。

建议考虑将原 Debian 基础镜像升级至 Photon-based Bitnami Secure Images。该版本与原 Debian 镜像兼容相同的热门 Helm 图表，升级后可获得以下优势：  
- 热门开源软件的强化安全镜像，实现近零漏洞  
- 漏洞分类与优先级划分，包含 VEX 声明、KEV 及 EPSS 评分  
- 聚焦合规性，支持 FIPS、STIG 及离线部署选项，提供安全物料清单（SBOM）  
- 通过 in-toto 提供软件供应链来源证明  
- 对主流 Helm 图表的一流支持
