# Java容器构建脚本仓库  


## 仓库用途  
本仓库包含用于生成官方 eclipse-temurin Java 容器的构建脚本。  


## Dockerfile 使用说明  
仓库内的 Dockerfile 供 [adoptium/containers]([]) 仓库调用，用于构建 Java 容器。  


## 平台支持  
当前仅支持 `linux/amd64` 平台。
