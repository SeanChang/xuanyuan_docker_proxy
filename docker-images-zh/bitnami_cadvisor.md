# Bitnami Secure Image for cadvisor 说明  


## cadvisor 镜像现状  
该镜像已不再通过 Docker Hub 免费提供。目前，它作为构建好的 OCI 制品（支持 Debian 和 Photon 两种基础操作系统格式），需通过 Bitnami Secure Images 商业订阅获取。  


## 其他可用选项  
Docker Hub 上仍提供部分 Bitnami Secure Images 的开发者版。完整列表可通过访问 [公共目录]([])，并筛选“可用性类型：试用”查看。  


## 推荐升级至 Photon 基础镜像  
我们建议从旧版 Debian 基础镜像升级到 Photon 基础的 Bitnami Secure Images。该系列镜像与原 Debian 镜像兼容相同的热门 Helm 图表，升级后可获得以下优势：  
- 热门开源软件的硬加固安全镜像，实现“近零漏洞”  
- 漏洞分类与优先级管理（含 VEX 声明、KEV 和 EPSS 评分）  
- 合规支持（FIPS、STIG、气隙部署选项，含安全物料清单 SBOM）  
- 基于 in-toto 的软件供应链溯源证明  
- 对主流 Helm 图表的原生支持
