# Bitnami Secure Image for milvus 介绍


## 镜像获取说明  
此镜像已不再通过 Docker Hub 免费提供。需通过 Bitnami Secure Images 商业订阅获取，提供 Debian 和 Photon 两种基础操作系统格式的预构建 OCI 制品。  
不过，Docker Hub 上仍提供其他 Bitnami Secure Images 开发者版。完整列表可通过[访问公共目录]([])并筛选“可用性类型：试用版”查看。  


## 推荐升级至 Photon-based 版本  
建议考虑将旧版 Debian 镜像升级至 Photon-based Bitnami Secure Images。该版本与原版 Debian 镜像兼容主流 Helm 图表，升级后可获得以下优势：  
- 热门开源软件的经过强化安全镜像，近零漏洞  
- 漏洞分类与优先级排序，提供 VEX 声明、KEV 和 EPSS 评分  
- 聚焦合规性，支持 FIPS、STIG 和离线部署选项，包含安全物料清单（SBOM）  
- 通过 in-toto 提供软件供应链来源证明  
- 对主流 Helm 图表提供一流支持
