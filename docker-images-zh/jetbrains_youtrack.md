# JetBrains YouTrack 官方 Docker 镜像介绍  


这是 [JetBrains YouTrack]  的官方 Docker 镜像，专为在 **Linux 环境** 中运行 YouTrack 设计。通过 Docker 使用时，它也完全兼容 macOS。该镜像包含在服务器环境或内部基础设施中部署 YouTrack 所需的所有组件，适用于评估场景和生产环境。  


## 安装与升级  
如需了解如何使用此镜像安装或升级 YouTrack，请参考 [官方文档] 。  


## 许可证  
本 Docker 镜像的使用需遵循 [《YouTrack 许可协议（服务器版，商业许可）》]  的条款。
