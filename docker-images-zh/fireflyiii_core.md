# Firefly III：免费开源的个人财务管理工具


## 简介  
Firefly III 是一款支持自托管的个人财务管理工具，免费开源。它能帮你记录收支明细，从而更合理地规划支出、增加储蓄。无论是想搞清楚钱花在哪儿，还是想提升财务安全感，这款工具都能为你提供支持。


## 核心功能  
- **收支跟踪**：记录每一笔收入和支出，清晰掌握资金流动。  
- **预算管理**：设置预算类别，控制开支不超支。  
- **分类与标签**：通过分类和标签对交易进行细化管理，方便筛选统计。  
- **数据导入**：支持从外部来源导入财务数据，减少手动录入工作量。  
- **财务报告**：提供多种可视化报告，直观展示财务状况和消费习惯。  


## 关于项目  
开发 Firefly III 的初衷很简单：让金钱不再成为焦虑的来源。无论是有钱还是没钱，付账单还是攒钱，金钱带来的压力常常让人困扰。这款工具希望帮你夺回财务主动权——无论账户余额多少，都能清晰掌控资金去向，从而感到踏实。开发者自己也在使用它，切实感受到了它带来的帮助，希望它也能为你提供支持。
