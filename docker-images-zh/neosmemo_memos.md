# Memos：自己搭的轻量笔记，数据握在自己手里  


## 项目简介  
Memos 是个开源的笔记应用，主打「自托管」—— 你可以把它部署在自己的服务器或本地设备上，所有笔记数据都存在自己手里，不用依赖第三方平台。适合日常记灵感、写待办、存碎片信息，或者团队内部做简单的知识库。  


## 核心特点  
- **数据自己管**：部署在自己的服务器/电脑上，笔记内容、用户信息都存在本地，不用担心平台关闭或数据泄露。  
- **轻量没广告**：安装包小，运行占资源少，界面干净，没有弹窗和广告，打开就能写。  
- **支持 Markdown**：标题、列表、代码块、链接这些常用格式都能直接写，编辑时实时预览效果。  
- **多端随时看**：手机、电脑、平板用浏览器打开就能访问，不用装额外 App，笔记自动同步。  
- **标签和搜索**：给笔记打标签分类，记多了也能按关键词快速找到旧内容。  


## 怎么装？  
### 推荐用 Docker 快速跑起来  
如果电脑装了 Docker，复制下面这行命令到终端，回车就能启动（首次运行会自动下载镜像）：  
```bash  
docker run -d --name memos -p 5230:5230 -v ~/.memos/:/var/opt/memos usememos/memos:latest  
```  
- `5230` 是访问端口，确保这个端口没被其他程序占用；  
- `~/.memos/` 是本地存储数据的文件夹，换成你自己的路径也行（比如 `/data/memos/`）。

## 日常怎么用？  
### 1. 新建笔记  
登录后点右上角「+」，输入标题和内容，支持直接写 Markdown（比如 `# 标题`、`- 列表`、`` `代码` ``），写完点「保存」就行。  

### 2. 给笔记打标签  
内容里写 `#标签名`（比如 `#工作`、`#灵感`），会自动生成标签，右侧能看到所有标签，点标签就能筛选同类笔记。  

### 3. 跨设备同步  
在手机浏览器输入服务器地址，登录后就能看/改笔记，内容实时同步（只要服务器没关）。  

### 4. 找旧笔记  
顶部搜索框输入关键词，会匹配标题和内容，结果秒出。
