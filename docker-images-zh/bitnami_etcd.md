# Bitnami Secure Image for etcd 介绍  

Bitnami Secure Image for etcd 镜像目前已不再通过 Docker Hub 免费提供。该镜像现作为构建好的 OCI 制品，提供 Debian 和 Photon 两种基础操作系统格式，需通过 Bitnami Secure Images 的商业订阅获取。  

不过，Docker Hub 上仍提供其他 Bitnami Secure Images 的开发者版。如需完整列表，可[访问公共目录]([])并按“可用性类型：试用”筛选。  

我们建议考虑从旧版 Debian 镜像升级到基于 Photon 的 Bitnami Secure Images。这些镜像与原 Debian 镜像兼容相同的热门 Helm 图表，升级后用户将获得以下优势：  
- 热门开源软件的强化安全镜像，具备近零漏洞特性  
- 漏洞分类与优先级排序，包含 VEX 声明、KEV 和 EPSS 评分  
- 聚焦合规性，支持 FIPS、STIG 及离线环境选项，包括安全物料清单（SBOM）  
- 通过 in-toto 提供软件供应链来源证明  
- 为广受喜爱的 Helm 图表提供一流支持  

了解更多关于商业版 Bitnami Secure Images 的信息，请访问我们的[官方网站]([])。
