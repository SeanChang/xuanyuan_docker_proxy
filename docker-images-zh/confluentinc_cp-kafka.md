# Apache Kafka 的 Confluent 社区版 Docker 镜像  


本 Docker 镜像用于部署和运行 Apache Kafka 社区版，基于 Confluent 社区版下载包构建。如需商业功能（仅包含在 [Confluent Server]  中），可参考 [cp-server]  镜像。
