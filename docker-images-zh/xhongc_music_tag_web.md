# 音乐标签 Web 版（Music Tag Web）

## 应用简介

「音乐标签 Web 版」（`xhongc/music_tag_web`）是一款运行在浏览器中的音乐标签（元数据）管理工具，适合部署在服务器或 NAS 上，通过 Web 界面批量整理音乐库。应用支持读取和编辑音频文件的标题、专辑、艺术家、风格、年份、封面图片等信息，并可配合 Navidrome 等音乐服务器作为“边车应用”使用。

## 主要功能

- **多格式音频支持**：支持 FLAC、APE、WAV、AIFF、WV、TTA、MP3、MP4、M4A、OGG、MPC、OPUS、WMA、DSF、DFF 等常见音频格式的元数据读取与写入。
- **可视化标签编辑**：通过 Web UI 批量查看和编辑音乐文件标签，支持搜索、筛选和排序。
- **批量处理能力**：一次性选择多个文件进行统一修改或规则化替换，提升大规模音乐库整理效率。
- **文件名与标签互相解析**：支持从文件名解析出艺术家/专辑/曲名等信息，也可以根据标签信息重命名文件。
- **封面管理**：支持查看、导出与替换专辑封面，方便修复错误封面或补充缺失封面。
- **繁简体转换与文本清洗**：提供繁简体转换和批量文本替换功能，用于清理“脏数据”与不规范标签。
- **适配移动端访问**：界面对移动端进行了适配，可通过手机或平板直接整理远程音乐库。

## 典型使用场景

- **远程音乐服务器配套工具**：与 Navidrome、Navidrome 兼容播放器等搭配，用于在服务器端直接整理音乐库标签。
- **大规模音乐库整理**：对已有的大量本地或远程音乐文件进行批量规范化处理（命名、标注、封面等）。
- **多终端协作整理**：将应用部署在局域网中，允许多台设备通过浏览器同时查看和调整音乐标签。

## Docker 部署示例

```bash
docker run -d \
  --name music-tag-web \
  -p 8081:8080 \
  -v /path/to/music:/music \   # 挂载音乐库目录
  -v /path/to/config:/config \ # 配置与缓存目录
  -e TZ=Asia/Shanghai \
  xhongc/music_tag_web:latest
```

启动后，在浏览器中访问 `http://<服务器IP>:8081` 即可打开 Web 管理界面，选择挂载的音乐目录进行扫描与标签整理。

## 数据与性能建议

- 对于大规模音乐库，建议在配置中开启增量扫描或分批处理，避免一次性全盘扫描造成开销过大。
- 建议定期备份配置目录和音乐库目录，以防误操作或磁盘故障导致数据丢失。
