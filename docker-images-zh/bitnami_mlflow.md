# Bitnami Secure Image for mlflow 说明


## 镜像获取说明  
Bitnami Secure Image for mlflow 已不再通过 Docker Hub 免费提供。目前，该镜像以构建好的 OCI 制品形式（支持 Debian 和 Photon 两种基础操作系统格式），通过 Bitnami Secure Images 商业订阅服务提供。  

不过，Docker Hub 上仍提供其他 Bitnami Secure Images 的开发者版本。如需查看完整列表，可访问 [公共目录]  并筛选“Availability Type: Trial”。  


## 推荐升级至 Photon-based 版本  
建议用户考虑从旧版 Debian 镜像升级至 Photon-based Bitnami Secure Images。该版本与原 Debian 镜像所使用的热门 Helm charts 完全兼容，升级后可获得以下优势：  


### 升级核心优势  
- **强化安全镜像**：基于流行开源软件构建，近零漏洞，安全性显著提升；  
- **漏洞管理优化**：支持通过 VEX 声明、KEV（已知被利用漏洞）和 EPSS（ exploit 预测评分系统）评分进行漏洞分类与优先级划分；  
- **合规能力增强**：聚焦合规需求，提供 FIPS、STIG 认证及离线部署选项，包含安全物料清单（SBOM）；  
- **供应链安全保障**：通过 in-toto 实现软件供应链来源证明，确保镜像完整性与可追溯性；  
- **Helm charts 支持**：对主流 Helm charts 提供一流适配与技术支持。
