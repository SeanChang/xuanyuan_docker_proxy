# Root Curated nginx

## 镜像概述
Root Curated nginx镜像是基于官方Docker [nginx](https://hub.docker.com/_/nginx)镜像精心打造的容器化应用基础镜像，旨在为用户提供安全、轻量级且便捷的应用部署起点。

## 核心功能和特性
Root Curated镜像具备以下核心优势：
- 减少镜像大小
- 最小化攻击面
- 改善初始安全态势

**注意**：若需实现零关键/高危漏洞，可考虑使用[Root Automated Vulnerability Remediation](https://app.root.io)。

## 使用场景和适用范围
