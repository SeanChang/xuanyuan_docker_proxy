# Grafana Enterprise Docker镜像  

Grafana Enterprise是Grafana的推荐发行版。无需许可证即可正常使用，且可轻松升级以启用企业级功能，例如数据源权限、报表功能及扩展认证选项。  


## 运行Grafana容器  

启动容器时绑定外部端口3000：  

```bash
docker run -d --name=grafana -p 3000:3000 grafana/grafana-enterprise
```
