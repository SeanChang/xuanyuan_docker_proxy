# Cass Config Builder

## 镜像概述和主要用途
Cass Config Builder是Apache Cassandra®的配置模板引擎，为通过`datastax/cass-operator`部署的容器提供配置支持。

## 贡献指南
自1.0版本起，Cass Config Builder由DataStax团队维护，是支持[DataStax Astra](https://www.datastax.com/cloud/datastax-astra)的组件之一。我们欢迎开源用户贡献错误报告、文档更新、测试和功能。
