# Lucky(大吉)

## 特性

Lucky 最初是开发者为个人使用而开发的小工具，用于在小米路由AX6000官方系统上替代socat，实现公网IPv6转内网IPv4的功能。其设计目标是让更多Linux嵌入式设备能够运行，集成个人用户常用功能，降低硬件和软件操作的学习成本，同时引导使用者关注网络安全。随着版本更新和网友反馈，Lucky 不断迭代，功能日益丰富，性能逐步提升，已成为一款值得信赖的工具。

Lucky 核心程序采用 Golang 开发，具备高效、稳定、跨平台的特性。后台前端基于 Vue3.2 构建，用户体验良好，响应迅速。管理后台采用前后端分离架构，第三方开发者可通过 OpenToken 轻松调用其各类功能接口。

## 功能模块

目前已实现/集成的主要功能模块包括：

- 端口转发
- 动态域名(DDNS)
- Web服务
- Stun内网穿透
- 网络唤醒
- 计划任务
- ACME自动证书
- 网络存储

### 端口转发
1.  核心功能是实现公网 IPv6 到内网 IPv4 的 TCP/UDP 端口转发。
2.  提供界面化管理转发规则，用户可通过 Web 后台便捷地添加、删除、修改规则。
3.  单条转发规则支持配置多个转发端口，满足多内网服务端口的转发需求。
4.  具备一键开关和定时开关功能，用户可按需设置规则的启用和禁用时间，也可结合计划任务模块实现定时控制。
5.  单条规则支持切换黑白名单安全模式。
6.  白名单模式有助于提升无安全验证的内网服务端口暴露到公网时的安全性，兼顾服务可用性。
7.  实时记录最新访问日志，方便用户掌握转发情况。
8.  规则列表与日志清晰展示，便于追踪转发异常并及时排查处理。

### 动态域名(DDNS)
1.  支持接入多家不同的 DNS 服务商。
2.  提供全功能自定义回调（Callback），包括 BasicAuth 设置，便于接入任意 DNS 服务商。
3.  Webhook 支持自定义请求头（headers）。
4.  内置常用免费 DNS 服务商设置模板（如每步、No-IP、Dynv6、Dynu），通过自定义回调可快速接入，用户只需修改相应的用户密码或 Token 即可一键填充配置。

### Web服务
1.  支持反向代理、重定向及 URL 跳转功能。
2.  提供 HTTP 基本认证机制。
3.  支持 IP 黑白名单模式。
4.  支持 UserAgent 黑白名单过滤。
5.  规则日志清晰，易于追踪异常访问。
6.  支持一键开关规则及定时开关规则。

### Stun内网穿透
1.  实现内网穿透功能，无需公网 IPv4 地址。
2.  适用于国内运营商提供的 NAT1 类型宽带网络。

### 网络唤醒
1.  支持远程控制设备唤醒和关机操作。
2.  可接入第三方物联网平台（如点灯科技、巴法云），并通过各大平台的语音助手控制设备的唤醒与关机。

### 计划任务
1.  不依赖 Linux 系统的 Cron 服务，支持 Windows 系统。
2.  操作简便，支持可视化编辑。
3.  可用于控制 Lucky 框架内其他模块的开关状态。

### ACME自动证书
1.  支持 ACME 协议的自动证书申请与续签。
2.  支持 Cloudflare、阿里云和腾讯云三大 DNS 服务商进行域名验证。

### 网络存储
1.  网络存储模块应用广泛，可将本地存储、WebDAV 及阿里云盘挂载到 Lucky 内部的各类文件服务中。
2.  通过该模块，用户添加的存储可被挂载到 Web 服务的文件服务、WebDAV、FTP 及 FileBrowser 模块，实现便捷的文件管理与访问。

对于 OpenWrt 系统用户，Lucky 提供了编译好的 IPK 安装包。用户可从官方发布页面下载与自己 OpenWrt 系统架构及版本相匹配的 IPK 包，通过 LuCI 界面或命令行工具（如 `opkg`）进行安装。

## 使用

成功安装 Lucky 后，默认情况下可通过访问设备的 IP 地址及默认端口（通常为 16601，具体可参考安装后的提示或官方文档）打开 Web 管理界面。首次登录需根据指引完成初始配置（如设置管理员密码），之后便可在 Web 界面中对各个功能模块进行配置和管理。
