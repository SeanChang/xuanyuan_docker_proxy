## 推荐升级至 Photon-based 版本  
建议从旧版 Debian 镜像升级到基于 Photon 的 Bitnami Secure Images。该版本与原 Debian 镜像使用的主流 Helm charts 完全兼容，升级后可获得以下优势：  
- 热门开源软件的强化安全镜像，近乎零漏洞  
- 漏洞分类与优先级排序支持，包含 VEX 声明、KEV 和 EPSS 评分  
- 聚焦合规需求，提供 FIPS、STIG 及离线部署选项，包含安全物料清单（SBOM）  
- 通过 in-toto 实现软件供应链来源证明  
- 对主流 Helm charts 的原生支持
