## 标签说明
openEuler 镜像标签采用其版本号命名，例如 `24.03-lts`、`22.03-lts-sp4`、`23.09` 等。


### 特殊标签
- `24.03` 或 `22.03`：分别代表对应 LTS 系列的推荐版本。  
- `latest`：表示最新的推荐版本。

## 关于 openEuler

![openeuler logo]([])  

openEuler 是一款开源免费的 Linux 发行版平台。它为全球开发者提供了开放社区，用于构建开放、多元且支持多架构的软件生态；同时也是鼓励创新的平台，欢迎大家提出新想法、探索新方法、实践新方案。  


### 操作系统
支持 Linux/Unix 系统，适配 ARM64、x86-64 或 loongarch64 架构。  


### 许可协议
openEuler 基于 Mulan PSL v2 协议发布，详情可查看仓库中的 `LICENSE` 文件。
