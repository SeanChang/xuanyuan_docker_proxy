# Hermes Web UI 镜像

Hermes 相关 Web 用户界面镜像，用于通过浏览器访问与管理 Hermes 服务或配套后端，提供对话、配置与监控等前端能力。

## 部署建议

- 与 Hermes API 或 Agent 后端同网络部署
- 配置反向代理与 HTTPS
- 根据官方文档设置后端地址与环境变量

适合需要自托管 Hermes 生态 Web 前端的开发与测试环境。
