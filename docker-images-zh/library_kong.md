## 什么是 Kong？  
Kong Gateway 是全球使用最广泛的 API 网关。作为轻量、高效、灵活的开源网关，它专为混合云和多云环境设计，且针对微服务和分布式架构优化。  

基于开源核心，Kong 的统一云 API 平台帮助从初创公司到财富 500 强的企业提升开发者生产力、保障构建安全，并加速产品上市。  

Kong 官方文档请访问 [docs.konghq.com]([])。
