File Browser 是一款文件管理工具，可在指定目录内提供文件管理界面。你可以用它上传、删除、预览、重命名和编辑文件。此外，它支持创建多个用户，每个用户可拥有独立的目录。

通过 Web 界面即可管理挂载目录中的文件，适合 NAS、内网文件共享或轻量文档协作场景。部署时通常将宿主机目录或 Docker 卷挂载到容器内指定路径，再映射 Web 端口供浏览器访问。
