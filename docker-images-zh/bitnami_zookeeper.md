## Bitnami Zookeeper 安全镜像说明  


### 镜像获取方式变更  
该 Zookeeper 安全镜像已不再通过 Docker Hub 免费提供。目前，它作为构建好的 OCI 制品（支持 Debian 和 Photon 两种基础操作系统格式），需通过 Bitnami 安全镜像的商业订阅获取。  

不过，Docker Hub 上仍提供其他 Bitnami 安全镜像的开发者版。完整列表可通过 [公共目录]([]) 筛选“Availability Type: Trial”查看。  


### 升级建议及优势  
建议考虑从旧版 Debian 镜像升级到基于 Photon 的 Bitnami 安全镜像。新镜像与原 Debian 镜像使用的热门 Helm 图表完全兼容，升级后可获得以下优势：  
- 流行开源软件的强化安全镜像，漏洞数量接近零；  
- 通过 VEX 声明、KEV 和 EPSS 评分实现漏洞分类与优先级排序；  
- 聚焦合规性，提供 FIPS、STIG 及离线部署选项，包含安全物料清单（SBOM）；  
- 通过 in-toto 提供软件供应链来源证明；  
- 对互联网主流 Helm 图表提供一流支持。
