# Teable 镜像文档

## 概述和主要用途
Teable 是一个超快速、实时、专业、开发者友好的无代码数据库，基于Postgres构建。它采用简单的类电子表格界面，用于创建复杂的企业级数据库应用，无需编写代码即可高效开发应用，同时避免数据安全和可扩展性的障碍。

## 核心功能和特性

### 类电子表格界面
包含所有您需要的功能：
- **单元格编辑**：直接点击并编辑单元格内容。
- **公式支持**：输入数学和逻辑公式自动计算值。
- **数据排序和筛选**：基于单列或多列排序数据；使用筛选器查看特定行数据。
- **聚合函数**：自动汇总各列统计数据，提供即时计算（如求和、平均值、计数、最大值、最小值），简化数据分析。
- **数据格式化**：格式化数字、日期等。
- **分组**：根据列值将行组织为可折叠组，便于数据分析和导航。
- **冻结列**：冻结表格左侧列，使其在滚动时保持可见。
- **导入/导出功能**：导入和导出其他格式数据（如.csv、.xlsx）。

## 使用方法和配置说明
### Docker 运行示例
假设镜像名称为 `teable/teable`，基本运行命令：
```bash
docker run -d -p 8080:8080 --name teable teable/teable
```
该命令将在后台运行Teable容器，映射容器的8080端口到主机的8080端口，通过 `http://localhost:8080` 访问应用。

### Docker Compose 配置示例（参考）
```yaml
version: '3'
services:
  teable:
    image: teable/teable
    ports:
      - "8080:8080"
    volumes:
      - teable_data:/data
    environment:
      - DATABASE_URL=postgres://user:password@postgres:5432/teable
    depends_on:
      - postgres

  postgres:
    image: postgres:14
    environment:
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=password
      - POSTGRES_DB=teable
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  teable_data:
  postgres_data:
```
> 注意：上述配置中的环境变量和端口映射可能需要根据实际情况调整，具体请参考Teable官方文档获取最新配置信息。
