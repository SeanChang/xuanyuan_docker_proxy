# Immich文件夹相册创建器

这是一个Python脚本，旨在从挂载到Immich容器的文件夹结构自动在[Immich](https://immich.app/)中创建相册。这对于为外部库自动创建和填充相册非常有用。通过提供的Docker镜像，该脚本可以轻松添加到Immich的compose栈中，与Immich的其他容器一起运行。
