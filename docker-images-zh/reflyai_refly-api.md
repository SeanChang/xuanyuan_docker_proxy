# Refly.AI

⭐️ 全球首个面向非技术创作者的 Vibe 工作流平台 ⭐️

Refly.AI 是全球首个 Vibe 工作流平台，赋能非技术创作者通过简单的提示词和可视化画布构建、分享和变现强大的 AI 自动化工作流——无需编写代码。

**面向非技术人员的 N8N**：像组装乐高积木一样轻松构建工作流。Refly.ai 赋能非技术创作者消除重复性任务，无需技术门槛。

**工作流的 Canva**：正如 Canva 让设计民主化，Refly.ai 让工作流创建民主化——让每个人都能轻松构建和分享自动化流程。

## ✨ 核心功能

### 可干预的智能体：消除不可预测的"黑盒"执行

**可视化执行**：工作流的每一步都在画布上清晰可见。便于轻松审查、理解和调试自动化流程。

**实时干预**：用户可以在执行过程中的任何时刻暂停、审查、修改或重启智能体。消除不稳定、不可逆或无法优化的"黑盒"执行带来的挫败感。

### 极简工作流工具：编排的不是节点，而是预打包的强大智能体

忘记繁琐的配置。在 Refly.ai 中，每个节点都是一个强大的、即用型智能体。您只需要分配任务并连接。Refly.ai 用两个节点在几分钟内完成 n8n 需要数小时和十个或更多节点才能完成的工作。

### 工作流副驾驶：将您的文字转化为复杂的自动化流程

描述一个任务，Refly 副驾驶立即在您的画布上创建、修改和调试复杂的自动化流程。即使是多步骤工作流也可以在几秒钟内构建完成。

### 工作流市场：一键发布，变现您的专业知识

**一键发布**：用户可以轻松将工作流转换为可分享的 AI 应用，并发布到工作流市场。

**变现您的专业知识**：创作者在用户每次运行其工作流时获得报酬，让您的经验和创作比以往更有价值。
