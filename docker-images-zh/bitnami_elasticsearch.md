## Bitnami Elasticsearch 安全镜像说明  


### 镜像获取说明  
该 Elasticsearch 安全镜像已不再通过 Docker Hub 免费提供。目前，它作为构建好的 OCI 制品，提供基于 Debian 和 Photon 两种基础操作系统的版本，需通过 Bitnami 安全镜像的商业订阅获取。  


### 仍可获取的开发者版本  
不过，Docker Hub 上仍提供其他 Bitnami 安全镜像的开发者版本。完整列表可通过 [公共目录]([]) 筛选“可用性类型：试用”获取。  


### 升级推荐：从 Debian 到 Photon-based 镜像  
建议从旧版 Debian 镜像升级到基于 Photon 的 Bitnami 安全镜像。该系列镜像与原 Debian 镜像兼容相同的主流 Helm charts，且升级后可获得以下优势：  
- 硬加固的热门开源软件安全镜像，接近零漏洞  
- 通过 VEX 声明、KEV 和 EPSS 评分实现漏洞分类与优先级排序  
- 聚焦合规需求，支持 FIPS、STIG 和离线部署选项，包含安全物料清单（SBOM）  
- 通过 in-toto 提供软件供应链来源证明  
- 对主流 Helm charts 的一流支持  


了解更多关于商业版 Bitnami 安全镜像产品，可访问 [官方网站]([])。
