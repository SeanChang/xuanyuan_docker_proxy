# Bitnami Secure Image for minio 说明  


### 镜像获取说明  
该镜像已不再通过 Docker Hub 免费提供。目前，它作为构建完成的 OCI 制品（支持 Debian 和 Photon 两种基础操作系统格式），需通过 Bitnami Secure Images 的商业订阅获取。  


### 开发者版镜像获取  
不过，Docker Hub 上仍提供其他 Bitnami Secure Images 的开发者版。如需查看完整列表，可访问公共目录并筛选“可用性类型：试用”。  


### 推荐升级至 Photon-based 镜像  
建议考虑从旧版 Debian 镜像升级到基于 Photon 的 Bitnami Secure Images。这类镜像与原版 Debian 镜像所适配的热门 Helm charts 完全兼容，升级后可获得以下优势：  
- 热门开源软件的强化安全镜像，具备近零漏洞特性  
- 通过 VEX 声明、KEV 及 EPSS 评分实现漏洞分类与优先级排序  
- 聚焦合规需求，支持 FIPS、STIG 及气隙部署选项，包含安全物料清单（SBOM）  
- 通过 in-toto 提供软件供应链溯源证明  
- 对主流 Helm charts 提供一流技术支持
